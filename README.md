# 🍗 Crispy Chikis  

**Crispy Chikis** is a mobile app for food delivery, inspired by modern restaurant platforms. Customers can order online, track their deliveries, and manage their profiles with ease.  

## 🌍 Features  
- 🛒 Order food online with a simple and fast process.  
- 💵 Cash payment available.  
- 📍 GPS tracking to follow delivery progress.  
- 👤 User profile to view past and current orders.
